A quem se destina?

Essa página é feita para ajudar quem tem seu projeto na cloud privada e quer utilizar a r3 da receita do splunk forwarder. Com ela, você pode gerar o inputs e o props para colocar na receita da sua aplicação ;)

Atenção! Nunca coloque o splunk para coletar um diretório inteiro. Caso existam muitos arquivos, podem ocorrer problemas com a sua indexação.


O index gerado no Inputs será raw_{sigla}. Por favor, assegure-se de que o index existe antes de qualquer coisa.


Coloque aqui de 3 a 5 eventos do seu log para que o app tente buscar na base de sourcetypes algum compatível.
